# WhatsApp Message Backup Viewer & Exporter

[![Expo](https://img.shields.io/badge/Expo-000020?style=for-the-badge&logo=expo&logoColor=white)](https://expo.dev/)
[![React Native](https://img.shields.io/badge/React_Native-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)](https://reactnative.dev/)

A premium, private, and powerful mobile application to view and export your WhatsApp chat backups locally. No data ever leaves your device.

## ✨ Features

- **Privacy First**: All processing is done locally on your device.
- **Multi-Language Support**: Available in English, Turkish, Arabic, German, French, Chinese, Spanish, and Russian.
- **Beautiful UI**: Clean, modern interface inspired by the best messaging apps.
- **Local File Support**: Upload `.txt` exports directly from WhatsApp.
- **Export Options**: Export your chats to organized formats for future reference.
- **Full Localization**: RTL support for Arabic users.
